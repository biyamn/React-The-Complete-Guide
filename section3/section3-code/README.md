# 내용 설명

아래와 같이 컴포넌트를 나누었습니다. Expenses, ExpenseItem, ExpenseDate 컴포넌트를 생성하였고 합성 컴포넌트 Card를 생성하여 Expenses, ExpenseItem에 적용하였습니다.
